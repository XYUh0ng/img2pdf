# Changelog

## v1.3.2 (2026-06-02)

### 🐛 Bug 修复

- **移动端侧边栏拖拽死锁**：修复 initTaskListSortable 未销毁旧实例导致多个 Sortable 监听器叠加，touchmove 手势死锁
- **弹窗触摸事件泄漏**：新增 touchcancel 处理，修复系统中断触摸（来电/通知）时 window 级监听器未清理

### 📝 文档

- 新增踩坑记录 docs/pitfalls/20260602-sortable-leak.md
- README 新增 docs/ 目录说明与移动端拖拽修复提示

## v1.3.1 (2026-06-01)

### 🐛 Bug 修复

- 修复弹窗缩放后任务列表拖拽失效（移除 window 级永久监听器，改为动态添加/移除）
- 修复移动端双指缩放松手后弹窗误关闭（新增 isPinching 状态）

## v1.3.0 (2026-06-01)

### ✨ 新功能

- **Ctrl+V 粘贴图片**：PC 端支持直接粘贴剪贴板中的图片（截图、网页图片等）
- **导出压缩选项**：导出 PDF 时可选压缩图片（默认开启），限制最大边长 2000px，JPEG 质量 0.8
- **原图缩放查看**：弹窗查看原图支持滚轮缩放（PC）和双指缩放（移动端）
- **缩放拖拽平移**：放大后可拖拽移动查看，含边界限制，原始尺寸不允许拖拽
- **任务列表删除按钮**：任务卡片直接删除，无需进入详情页

### 🐛 Bug 修复

- 修复移动端任务列表拖拽排序不生效（SortableJS onEnd → onUpdate）
- 修复移动端弹窗触摸事件干扰其他组件
- 修复删除按钮阻断 SortableJS 触屏拖拽

## v1.2.0 (2026-06-01)

### ✨ 新功能

- **移动端响应式适配**：sidebar 改为左侧抽屉（汉堡按钮触发），editor 改为上下布局
- **触屏优化**：删除按钮常显、触摸区加大 44px、上传区简化文案
- **移动端禁用悬浮预览**：避免触屏设备无效的 hover 交互

### 🔧 改进

- 工具栏 flex-wrap 自适应换行，小屏不溢出
- 图片网格移动端固定 2 列，卡片高度缩小
- 顶栏标题省略号截断，不换行

## v1.1.0 (2026-06-01)

### ✨ 新功能

- **悬浮预览**：鼠标悬浮缩略图自动显示 2 倍放大预览，智能定位（左右/上下自适应），支持开关切换
- **原图查看**：点击缩略图弹窗查看原图，支持点击背景/ESC/× 按钮关闭
- **任务列表拖拽排序**：左侧任务列表支持拖拽调整顺序
- **数据持久化**：图片数据使用 IndexedDB 持久化存储，关闭浏览器后重新打开任务和图片不丢失

### 🔧 改进

- 存储层重构：localStorage 仅存任务元数据，图片 dataUrl 改用 IndexedDB（容量远大于 localStorage 的 ~5MB 限制）
- 外部依赖改为本地引入（lib/ 目录），解决 file:// 协议下 CDN 加载失败问题
- 新增 `下载依赖库.bat` 一键下载脚本和 `setup.ps1` PowerShell 脚本
- 拖拽排序时悬浮预览不触发，拖拽后不误触弹窗

## v1.0.0 (2026-06-01)

### 🎉 首个版本

- 批量上传图片（JPG / PNG / BMP / WEBP）
- 图片缩略图预览与拖拽排序
- 多任务管理（新建 / 切换 / 删除）
- 任务自定义命名，导出 PDF 文件名为任务名
- 一键导出 A4 尺寸 PDF
- 纯前端实现，零后端依赖
